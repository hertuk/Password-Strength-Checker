# 🔑 Password Strength Checker

A simple password strength analyzer implemented in multiple programming languages.

---

## 📖 Overview

This project demonstrates password validation and strength checking using:

- JavaScript
- Python
- C#
- PHP

Useful for:

- Authentication systems
- Registration forms
- Security learning
- Utility tools

---

## ✨ Features

- Password length validation
- Uppercase character detection
- Number detection
- Simple strength evaluation
- Multi-language implementation
